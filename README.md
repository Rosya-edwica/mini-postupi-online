## Почему mini?

Потому что парсер собирает только ограниченное количество данных.

- аббревиатура
- название
- город

#### Специальность:
- код
- название
- направление

#### Программа:
- название
- уровень
- форма обучения
- платное обучение - да или нет
- стоимость обучения
- количество бюджетных мест
- количество платных мест
- Предметы ЕГЭ или вступительные экзамены
